## üöÄ Installation Guide (Docker / TrueNAS)

### Prerequisites

Ensure you have the following installed on your host (TrueNAS Scale, Ubuntu, etc.):
* **Docker Engine**
* **Docker Compose**

### Quick Setup

1.  **Clone the Repository:**
    ```bash
    git clone -b truenas_hosting [repository_url] StreamForge
    cd StreamForge
    ```

2.  **Environment Configuration:**
    * Navigate to the `backend/` directory.
    * Create a `.env` file and add your API keys:
    ```bash
    GOOGLE_API_KEY=your_api_key_here
    SECRET_KEY=your_secret_key
    ```

3.  **Build & Deploy:**
    Run the following command in the root directory (where `docker-compose.yml` is located):
    ```bash
    docker-compose up -d --build
    ```

4.  **Access the Platform:**
    * **Frontend**: Navigate to `http://<NAS-IP>:8080` (e.g., `http://192.168.1.100:8080` or `http://localhost:8080`).
    * **Backend API**: Running on `http://<NAS-IP>:5000`.

### üìÇ Data Persistence

This setup uses Docker volumes to ensure your data survives container restarts. The following folders will be created in your project root:

* `./data/pipelines.db` - Stores the SQLite database.
* `./data/uploads` - Stores raw user-uploaded files.
* `./data/processed` - Stores result files generated by pipelines.

## üîí Credentials & Authentication

StreamForge utilizes **JWT Authentication**.

* **New Users**: Register via the **Sign Up** page at `http://<NAS-IP>:8080/signup`.
* **Admin Access**: The first registered user acts as the Administrator by default (configurable in `backend/app/models.py` if needed).
